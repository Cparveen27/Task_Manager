# Task_Manager Web Application

A modern and responsive Task Manager Web Application built using HTML, CSS, and JavaScript.
This app helps users manage daily tasks efficiently with features like dark mode, search, and task categorization.



## 📌 Features

- ✅ Add, edit, and delete tasks
- 📅 Add due dates to tasks
- 🔍 Search tasks in real-time
- 🌙 Dark mode & ☀️ Light mode toggle
- 📂 Task categories (Study, Work, Personal)
- ✔️ Mark tasks as completed
- 💾 Data stored using localStorage (no backend required)
- 🎨 Modern UI with clean design



## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
